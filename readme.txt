Zero Parallel Affiliate network form

== Description ==

Zero Parallel Affiliate network form integration

Simple plugin that allows you to integrate Loan Form on your wordpress site.
Just use Form button in wordpress editor to make it work.
You must register in https://zeroparallel.com/ to earn money.

Requires WordPress 4.1 and PHP 5.

== Installation ==

1. Upload all files to the `/wp-content/plugins/zero-parallel-form/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use short tag [zp-form user="YOUR_AFFILIATE_ID"] in your posts and pages
